# Getting Started

Install fledge and get started in about two minutes — whether you're creating a new project or adding fledge to an existing one.

1. **[Installation](getting-started/installation.md)** — Install via crates.io, Homebrew, Nix, or the curl installer.
2. **[Quick Start](getting-started/quick-start.md)** — Create a new project from a template and run your first tasks.
3. **[Existing Projects](getting-started/existing-projects.md)** — Add fledge to a repo you already have — most features work with zero config.
